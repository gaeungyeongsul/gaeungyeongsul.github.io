1. 두개의 정수와, 한개의 사칙연산을 입력받아서
계산의 값을 출력하세요.
```Java
Scanner input = new Scanner(System.in);
System.out.println("정수값을 입력하세요 : ");
int a = input.nextInt();
System.out.println("정수값을 입력하세요 : ");
int b = input.nextInt();
System.out.println("사칙연산값을 입력하세요 : ");
String str = input.next();
switch (str) {
		case "+" :
			System.out.println(a+b);
			break;
		case "-" :
			System.out.println(a-b);
			break;
		case "/" :
			System.out.println(a/b);
			break;
		case "*" :
			System.out.println(a*b);
			break;
}
```

2. 초 단위 시간을 입력받아서 시, 분, 초 단위로 변환하는 프로그램.
```Java
Scanner sc = new Scanner(System.in);
System.out.println("초를 입력하세요 : " );
int sec = sc.nextInt();
int hour = sec/3600;
int min = (sec-3600*hour)/60;
int second = (sec-3600*hour-60*min);
System.out.printf("%d시간 %d분 %d초 입니다.", hour, min, second);
```

3-2. 피보나치 수열에서 사용자가 입력한 정수(n) 까지의 합을 출력하세요.
```Java
Scanner input = new Scanner(System.in);
System.out.println("정수를 입력하세요 : ");
int n = input.nextInt();
int a = 0;
int b = 1;
int fibo = 0;
int sum = 1;
for(int i = 0; i < n-2; i++) {
	fibo = a + b;
	sum = fibo + sum;
	a = b;
	b = fibo;
}
System.out.println(sum);
```
4. 3개의 For 문을 이용해서 구구단(1~9단) 을 출력하세요<br>
   1단  2단  3단<br>
   4단  5단  6단<br>
   7단  8단  9단<br>
```Java
for(int i = 1; i<=3; i++) {
  for(int j = 1; j <= 9; j++) {
		for(int k = 3*i-2; k<=3*i; k++) {
			System.out.print(k + "X" + j + "=" + k*j + " ");
			}System.out.println();
	}
}
```
